# EcoTrack AI 🌱

An AI-powered Carbon Footprint Awareness Platform developed for Virtual Challenge 3.

## Features

- Calculate estimated monthly carbon footprint
- AI-generated sustainability recommendations
- Carbon footprint category detection (Low, Medium, High)
- 7-Day Eco Challenge for users
- Clean and responsive UI
- Instant results without page reload

## Technologies Used

- HTML
- CSS
- JavaScript

## How It Works

Users enter:
- Monthly electricity usage
- Monthly travel distance
- Food preference (Veg / Non-Veg)

The system estimates carbon footprint and provides personalized eco-friendly suggestions.
